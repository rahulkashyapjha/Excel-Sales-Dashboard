# VIFAO Company Performance Dashboard (Excel Project)

## Project Overview

This project is an interactive Excel dashboard that analyzes company performance including sales, profit, customers, and product categories.

The dashboard helps to quickly understand business insights using charts, KPIs, and slicers.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Dashboard Design

## Key Metrics in Dashboard

* Total Sales
* Total Profit
* Total Customers
* Total Countries
* Total Categories

## Dashboard Insights

* Sales and Profit trend over time
* Top 10 customer base
* Sales by gender and category
* Order processed by days of week
* Contribution of each category to total profit
* Company product contribution

## Features

* Interactive Slicers
* Dynamic Charts
* KPI Cards
* Clean Dashboard Layout

## Skills Demonstrated

* Data Analysis
* Data Visualization
* Excel Dashboard Building
* Business Insights
